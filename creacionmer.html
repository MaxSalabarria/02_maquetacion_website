Creación de mercancía

Descripción de proyecto

Una tienda con el nombre "Diseños Maxi" que se dedica a la compra y venta de productos de papelería requiría unos diseños para
lo que sería su mercancía de su marca, en este caso los más básicos como libretas, gorras, lápices, plumas, etc.
Esto es tanto para mostrarle a lo clientes los productos que quieran personalizar o comprar la que será de la tienda.
Un punto importante en la realización de los diseños es que todos deben llevar el logo.

>Lo que se hizo en el proyecto

Para este proyecto se utilizó el software de dibujo Krita, para la creación de diseños, ha pasado por varios chequeos, pero dichos
diseños fueron un éxito, a continuación se mostrarán los más destacados.

  1. Libreta

    Libreta que será para venta a los clientes, como una forma de mostrarnos a más público.

  2. Mousepad

    Mousepad que será para venta a clientes.

  3. Taza

    Taza que será para venta y quizás su los empleados deseen pueden llevar una.

  4. Gorra

    Gorra que será para venta.

  5. Camisa

    Camiseta que será para venta a los clienntes, los empleados deben llevar puesto esta camisa como uniforme.

  6. Mochila

    Esta mocila será para venta para los clientes
